
# Kelvin's Website

## Getting Started

### Commands

Install dependencies.

`npm install`

Serve with hot reload at <http://localhost:5173>.

`npm run dev`

### Lint

`npm run lint`

### Typecheck

`npm run typecheck`

### Build

`npm run build`

### Test

`npm run test`

View and interact with your tests via UI.

`npm run test:ui`
